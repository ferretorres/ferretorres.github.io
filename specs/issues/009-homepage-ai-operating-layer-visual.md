# Build: Homepage AI operating layer visual

## Problem

The homepage needs an immediate visual signal that Ferre Torres B.V. implements AI systems across data, workflows, agents, dashboards, and business tools.

## User story

As a first-time visitor, I want to understand the AI-first operating layer quickly, so I can decide whether the company is relevant for my transformation project.

## Scope

- Add a visual section near the top of `index.html`.
- Show data sources flowing into a Company Brain / AI operating core.
- Show RAG, agents, dashboards, approval workflows, and business systems as outputs.
- Keep it professional, responsive, and not overly decorative.
- Link visual nodes to relevant pages where appropriate.

## Acceptance criteria

- Homepage has a clear operational visual in the first half of the page.
- Visual supports the "AI consultancy for companies" positioning.
- It does not make the page feel like a generic SaaS landing page.
- Mobile layout is clean and readable.

## Verification

- Browser desktop and mobile checks.
- Check text contrast and no overlap.
- Confirm primary CTA remains visible and clear.
