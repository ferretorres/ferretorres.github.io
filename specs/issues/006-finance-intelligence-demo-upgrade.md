# Upgrade: Finance intelligence guided demo visual

## Problem

Finance intelligence is commercially strong, but the current page needs visuals that show how it differs from static BI dashboards.

## User story

As a CFO or finance lead, I want to see how AI explains margins, pricing, forecasts, and anomalies, so I can understand the value beyond Tableau or PowerBI dashboards.

## Scope

- Upgrade `demo-finance-intelligence.html`.
- Add a synthetic finance intelligence dashboard with KPIs, variance explanation, pricing signals, margin movement, anomaly detection, and recommended follow-up actions.
- Include labels such as dynamic price spots, trend detection, no dependency on static BI dashboards, forecast explanation, and approval workflow.
- Add questions for scoping finance data and expected usage.

## Acceptance criteria

- Page clearly positions finance intelligence as an AI-first operating tool, not just a dashboard.
- Synthetic values are visually credible and clearly fake.
- CTA invites a guided demo meeting.
- The page remains fast and static.

## Verification

- Browser desktop and mobile checks.
- Visual text fit check.
- Static metadata check.
