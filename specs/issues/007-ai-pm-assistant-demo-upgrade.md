# Upgrade: AI PM assistant guided demo visual

## Problem

The AI PM assistant page should show how project context, risks, decisions, timelines, and stakeholder updates are handled.

## User story

As an operations or product leader, I want to see how an AI PM assistant helps with delivery control, so I can understand how it reduces coordination load.

## Scope

- Upgrade `demo-ai-pm-assistant.html`.
- Add a synthetic project control cockpit with milestones, risk alerts, meeting memory, action owners, blockers, and next-step drafts.
- Show human approval for stakeholder communications.
- Add scoping questions about project volume, tools, users, and governance.

## Acceptance criteria

- Visual preview shows project intelligence, not generic productivity.
- Page explains where AI drafts, where humans approve, and where actions are tracked.
- CTA is clear and above/below the demo section.

## Verification

- Browser desktop and mobile checks.
- Internal links pass.
- No false claims or client data.
