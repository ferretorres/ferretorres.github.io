# Upgrade: Agentic workflow guided demo visual

## Problem

Agentic AI can sound risky or vague. The site should show controlled agent workflows with approvals, tool access, audit trails, and escalation.

## User story

As a COO, CTO, or compliance-aware buyer, I want to see how agents can operate safely inside real workflows, so I can evaluate where automation is appropriate.

## Scope

- Upgrade `demo-agentic-workflow.html`.
- Add a synthetic workflow control panel showing task routing, tool calls, confidence, approval gates, exceptions, and audit log.
- Show agents as controlled workflow participants, not fully autonomous black boxes.
- Add scoping questions about tools, permissions, user roles, and escalation rules.

## Acceptance criteria

- The page makes agentic workflow safety visually clear.
- Approval gates and audit trails are prominent.
- CTA invites guided demo meeting.
- Mobile view remains polished.

## Verification

- Browser desktop and mobile checks.
- Static metadata check.
- Confirm no horizontal overflow.
